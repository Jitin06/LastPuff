Step 1
Create a Gmail App Password
You need a special App Password (not your regular Gmail password).

1. Go to myaccount.google.com/security
2. Enable 2-Step Verification if not already on
3. Search for "App Passwords" in the search bar
4. Create a new app password — name it Last Puff
5. Copy the 16-character password shown (e.g. abcd efgh ijkl mnop)
Step 2
Enter your Gmail details
Credentials are saved locally on your laptop only. Never sent anywhere.

How it works
• The app at index.html automatically syncs your data to this server every time you log a cigarette

• Every night at 11:59 PM IST, the server sends a beautiful HTML report to your email

• Keep this terminal window open (or run it in the background) for emails to work

• You can also click "Send Test Email Now" to verify it's working
